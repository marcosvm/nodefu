We do not currently have a full featured website as we are focusing on the API and the service. We are currently running Node v.0.3.5 and updating all NPM modules on a weekly basis. Git is required to push your updates to Nodefu. All we ask is that you be a good steward of our service and community.

COUPON

Request Coupon :: POST
Base URL: http://nodefu.com

/coupon - creates coupon request for early access (pass in email)

curl -X POST -d "email=your_address@gmail.com" http://nodefu.com/coupon

STATUS

Get Status :: GET
Base URL: http://nodefu.com

/status - Returns platform status and number of apps running

curl http://nodefu.com/status

USER

Register User :: POST (Coupon is required.)
Base URL: http://nodefu.com

/user - creates user account (pass in user and password and email and id_rsa.pub string)

curl -X POST -d "user=testuser&password=123&email=chris@nodefu.com&rsakey=ssh-rsa AAAAB3NzaC1yc..." http://nodefu.com/user

Delete User :: DELETE
Base URL: http://api.nodefu.com

/user - delete user account (requires basic auth)

curl -X DELETE -u "testuser:123" http://api.nodefu.com/user

APP

Create Application :: POST
Base URL: http://api.nodefu.com

/apps - create nodejs app for hosting (requires basic auth and returns the port address required for use along with a git repo to push to)

curl -X POST -u "testuser:123" -d "appname=a&start=hello.js" http://api.nodefu.com/app

Change Application :: PUT
Base URL: http://api.nodefu.com

/apps - update nodejs app for hosting (requires basic auth, appname, and starting page and returns the port address required for use along with a git repo to push to)

curl -X PUT -u "testuser:123" -d "appname=a&start=hello1.js" http://api.localhost:8080/app

Delete Application :: DELETE
Base URL: http://api.nodefu.com

/apps - delete nodejs app (requires basic auth and appname)

curl -X DELETE -u "testuser:123" -d "appname=test" http://api.nodefu.com/app

Application information :: GET
Base URL: http://api.nodefu.com

/app/appname - get app information (requires basic auth and appname)

curl -u "testuser:123" http://api.nodefu.com/app/appname

GIT

Deploying and updating your Node.js application is simple.

curl -X POST -u "testuser:123" -d "appname=myapp&start=hello.js" http://api.nodefu.com/app

Upon creating or changing your application via our API, you will receive a Git reop URL from our API response. Add a NodeFu remote to your project as follows:

git remote add nodefu the_url_returned_by_our_api

Finally push your updates to your new NodeFu environment as follows:

git push nodefu master

Also make sure that you update your project's port address with the new one assigned by NodeFu.

Visit your application via http://myapp.nodefu.com
